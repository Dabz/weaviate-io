[Weaviate Cloud (WCD)](https://console.weaviate.cloud/) provides a web interface and an API endpoint that provide information about your Weaviate instance.

## Status through Weaviate Cloud console

When you log into the Weaviate Cloud console, the left sidebar lists your clusters. When you click on a cluster, you can see the following fields in the `Cluster details` section at the top of the page:

- `Dimensions stored`
- `Dimensions queried`
- `Object count`
- `Running costs`

The following fields are part of the `Information` section:

- `Name`
- `REST Endpoint`
- `gRPC Endpoint`
- `Weaviate Database version`
- `Cloud region`
- `Created at`
- `Type`: Serverless or Sandbox
- `Service Level Agreement`
- `High Availability`

Serverless clusters also have an `Advanced configuration` section to configure or to check the status of these additional settings:

- `Enable auto schema generation`
- `Enable async indexing`
- `Allow all CORS origins`
